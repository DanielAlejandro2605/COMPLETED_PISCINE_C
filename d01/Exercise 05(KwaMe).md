# day 01 Exercise 05

 - Создайте файл "\?$*'KwaMe'*$?\" , содержащий только «42» и НИЧЕГО больше.

---
---
---



Создаем файл через программу еcho c помощью перенаправления потока вывода в файл вот так:
```
echo -e "42" | tr -d "\n" > '"\?$*'KwaMe'*$?\"'
```


или так:
```
echo -e "42" > '"\?$*'KwaMe'*$?\"'
```
